# 🚀 FYP Hub — Collaboration Space

FYP Hub is a modern collaboration workspace built for Final Year Project teams to manage their project progress, members, research papers, meetings, shared files, and communication in one place.

It provides a clean and interactive dashboard that helps student groups stay organized throughout the entire project lifecycle.

---

## ✨ Features

### 🔐 Member Login & Access
- Sign in using a registered email address
- First-time member setup and registration
- Avatar color selection for each team member
- Session-based login handling

### 📊 Project Dashboard
- Overview of project progress
- Project title, description, supervisor, and batch details
- Progress bar based on completed tasks
- Quick view of:
  - Recent tasks
  - Upcoming milestones
  - Research papers
  - Meeting notes
  - Recent uploads

### ✅ Task Management
- Kanban-style workflow
- Track tasks as:
  - To Do
  - In Progress
  - Done
- Assign tasks to team members
- Priority-based organization
- Due date tracking

### 📅 Timeline & Milestones
- Add and manage project milestones
- Monitor deadlines
- Track completed and pending progress stages

### 📚 Research Management
- Store and manage research papers
- Mark papers by reading status
- Keep references organized for literature review and project development

### 📝 Meetings Section
- Save meeting notes
- Track discussions and decisions
- Maintain project meeting history

### 💬 Team Chat
- Built-in team communication section
- Send and view group messages
- Supports simple internal collaboration

### 📁 File Sharing
- Upload and manage shared files
- Google Drive integration for file storage
- Recent uploads displayed on dashboard

### 👥 Member Management
- Add new group members
- View registered team members
- Remove members when needed

### 🎨 Modern UI/UX
- Responsive dark-themed interface
- Stylish sidebar navigation
- Smooth transitions and clean layout
- Designed for project collaboration and productivity

---

## 🧩 Tech Stack

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript

### Backend / Storage Services
- JSONBin API for app data persistence
- Google Apps Script Web App for file upload handling
